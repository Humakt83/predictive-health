# Backend

Using python flask 

* Python 3.8.0

* ```
  pip install Flask
  ```

* ```
  pip install Flask-JSON
  ```

  

  

## On windows machine:

***>cd to <repo_path>\predictive-health\backend***

***>set FLASK_APP=%cd%\api\predictive.py***

***>set FLASK_ENV=development***

***>flask run***
